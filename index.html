INTRODUÇÃO

Este projeto foi criado para a disciplina de Visualização de Dados do Centro de Informática (CIn - UFPE). O mesmo teve como objetivo criar uma ferramenta que permitisse ao usuário visualizar informações de uma forma mais atrativa e de fácil interpretação.


Com isso, esta ferramenta foi criada para mostrar a realidade dos estudantes do Ensino Médio do Estado de Pernambuco que prestam vestibular para a Universidade de Pernambuco (UPE). Nela será possível identificar relações entre as notas obtidas nas provas e região, as principais escolhas por região, bem como identificar em quais regiões os estudantes se concentram.
